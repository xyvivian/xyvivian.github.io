# xyvivian.github.io

Academic personal site for Xueying Ding, built with Jekyll and designed for GitHub Pages.

## Local development

1. Install Ruby and Bundler if needed.
2. Run `bundle install`.
3. Run `bundle exec jekyll serve`.
4. Open `http://127.0.0.1:4000`.

## Content updates

- Profile and hero content lives in `_data/profile.yml`.
- Publications live in `_data/publications.yml`.
- Styles live in `assets/css/style.scss`.

Before publishing, replace the placeholder LinkedIn and Google Scholar URLs in `_data/profile.yml`.
